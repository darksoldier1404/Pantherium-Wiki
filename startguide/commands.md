# 🧾 명령어 정리

### ✈️ 이동 관련 명령어

| 명령어         | 설명                      |
| ----------- | ----------------------- |
| `/tpa 닉네임`  | 해당 플레이어에게 텔레포트를 요청합니다.  |
| `/tpaccept` | 상대방이 보낸 텔레포트 요청을 수락합니다. |
| `/tpdeny`   | 상대방이 보낸 텔레포트 요청을 거절합니다. |

> 예시: `/tpa panterium`

***

### 🕊️ 편의 기능 명령어

| 명령어     | 설명                    |
| ------- | --------------------- |
| `/tfly` | 플라이 모드를 켜거나 끌 수 있습니다. |

> 사용 시 현재 상태에 따라 ON / OFF가 전환됩니다.

***

### 🏝️ 섬 팀 관련 명령어

| 명령어                           | 설명                     |
| ----------------------------- | ---------------------- |
| `/island team invite`         | 플레이어를 자신의 섬에 초대합니다.    |
| `/island team accept`         | 받은 섬 초대를 수락합니다.        |
| `/island team reject`         | 받은 섬 초대를 거절합니다.        |
| `/island team leave`          | 현재 속한 섬 팀에서 나갑니다.      |
| `/island team kick [닉네임]`     | 해당 팀원을 섬에서 추방합니다.      |
| `/island team setowner [닉네임]` | 섬의 소유권을 해당 팀원에게 이전합니다. |

***

### 🏠 섬 홈 관련 명령어

| 명령어                       | 설명                                             |
| ------------------------- | ---------------------------------------------- |
| `/island sethome [이름]`    | 섬의 홈(귀환 지점)을 설정합니다. 이름을 지정하여 여러 홈을 관리할 수 있습니다. |
| `/island deletehome [이름]` | 지정한 이름의 섬 홈을 삭제합니다.                            |
| `/island rename [이름]`     | 섬 홈 이름을 변경합니다.                                 |

***

### 🔄 섬 초기화 명령어

| 명령어         | 설명             |
| ----------- | -------------- |
| `/is reset` | 자신의 섬을 초기화합니다. |

> ⚠️ 섬 초기화는 되돌리기 어려울 수 있으니 신중하게 사용해주세요.

***

### 📌 안내사항

* 일부 명령어는 특정 상황이나 권한에 따라 사용이 제한될 수 있습니다.
* 명령어 사용이 되지 않는 경우 서버 내 안내 또는 공지를 먼저 확인해주세요.
* 추가 명령어는 업데이트에 따라 변경되거나 추가될 수 있습니다.
