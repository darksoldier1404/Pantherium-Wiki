# 판테리움 서버 위키

## 🌿 환영합니다

판테리움 서버 위키에 오신 것을 환영합니다.\
판테리움은 **생활 콘텐츠**, **성장 시스템**, **섬 운영**, **경제 활동**이 자연스럽게 연결된 감성 마인팜 서버입니다.

처음 오신 분도 쉽게 적응할 수 있도록, \
이 위키에서는 서버의 핵심 정보와 주요 시스템을 한눈에 확인하실 수 있습니다.

***

### 🖥️ 서버 접속 정보

| 항목         | 내용                      |
| ---------- | ----------------------- |
| **서버 주소**  | `panterium.kr`          |
| **권장 버전**  | `1.21.4`                |
| **플레이 방식** | 생활 · 성장 · 섬 운영 · 경제 콘텐츠 |

> 💡 꼭 서버 접속 전에 서버 편집에서 **서버 리소스팩**을 사용으로 하고 접속해주세요

***

### 📖 판테리움은 어떤 서버인가요?

자신만의 섬을 성장시키고, 생활 콘텐츠를 통해 재화를 모으며,\
다양한 시스템을 활용해 꾸준히 성장해나갈 수 있습니다.

***

### ⚠️ 이용 전 꼭 확인해주세요

* 서버 규칙과 운영 정책은 반드시 확인해주세요.
* 권장 버전이 아닐 경우 접속 또는 플레이가 원활하지 않을 수 있습니다.
* 일부 시스템은 업데이트에 따라 변경될 수 있습니다.
* 공지사항 및 문의 안내는 디스코드를 통해 가장 빠르게 확인하실 수 있습니다.

> 🚨 원활한 플레이를 위해 접속 전 공지와 안내사항을 꼭 확인해주세요.
