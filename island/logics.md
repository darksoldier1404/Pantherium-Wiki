# 🧩시스템 설명

***

### 🌤️ 섬 시스템이란?

판테리움의 섬은 단순히 머무르는 공간이 아닙니다.\
플레이를 이어가면서 여러 콘텐츠가 자연스럽게 섬과 연결됩니다.

* 생활 콘텐츠를 위한 기본 활동 공간
* 자원을 모으고 정리하는 성장 공간
* 나만의 스타일로 꾸밀 수 있는 공간
* 팀원과 함께 운영할 수 있는 협동 공간
* 다른 유저가 방문하고 반응을 남길 수 있는 교류 공간

즉, 섬은 판테리움 플레이 전반을 이어주는 중심 공간이라고 볼 수 있습니다.

***

#### 👥 팀원과 함께 운영하기

판테리움의 섬은 팀 플레이도 가능합니다.\
섬 팀은 **본인을 포함해 최대 4명까지** 함께할 수 있으며, 팀원을 초대해 하나의 섬을 같이 성장시키고 운영할 수 있습니다.

**팀 관련 주요 명령어**

| 명령어                           | 설명                     |
| ----------------------------- | ---------------------- |
| `/island team invite`         | 플레이어를 자신의 섬에 초대합니다.    |
| `/island team accept`         | 받은 섬 초대를 수락합니다.        |
| `/island team reject`         | 받은 섬 초대를 거절합니다.        |
| `/island team leave`          | 현재 속한 섬 팀에서 나갑니다.      |
| `/island team kick [닉네임]`     | 해당 팀원을 섬에서 추방합니다.      |
| `/island team setowner [닉네임]` | 섬의 소유권을 해당 팀원에게 이전합니다. |

#### 📈 성장 결과 확인하기

섬을 발전시키고 다양한 요소를 채워나갈수록 섬의 가치와 성장도가 높아지게 됩니다.

**레벨 관련 주요 명령어**

| 명령어         | 설명               |
| ----------- | ---------------- |
| `/is level` | 현재 섬의 레벨을 확인합니다. |

> 서버 설정에 따라 표시 방식이나 접근 방법은 달라질 수 있습니다.

#### 🚪 방문과 교류

워프를 열어 다른 유저가 내 섬을 방문하게 하거나, 좋아요 기능을 통해 반응을 받을 수 있습니다.

해당 사진과 같이 표지판으로 \[워프] 라고  적게되시면 다른 사람들이 자신의 섬에 방문이 가능합니다.&#x20;

**워프 관련 주요 명령어**

| 명령어        | 설명                              |
| ---------- | ------------------------------- |
| `/is warp` | 섬 워프 기능을 확인하거나 워프 관련 설정을 진행합니다. |

**좋아요 관련 안내**

좋아요 기능은 다른 유저가 내 섬에 반응을 남길 수 있는 시스템입니다.\
섬을 꾸미고 공개하는 재미를 더해주는 요소로 활용됩니다.

다른 플레이어의 섬에 가서 섬메뉴를 이용해 좋아요를 남길수 있습니다.

> 좋아요 기능은 서버 설정에 따라 메뉴 또는 UI를 통해 이용할 수 있습니다.

#### 🏠 홈 설정과 관리

섬 안에서 자주 이동하는 위치는 홈으로 지정해두면 더 편리하게 이용할 수 있습니다.

**홈 관련 주요 명령어**

| 명령어                       | 설명                  |
| ------------------------- | ------------------- |
| `/island sethome [이름]`    | 섬의 홈(귀환 지점)을 설정합니다. |
| `/island deletehome [이름]` | 지정한 이름의 섬 홈을 삭제합니다. |
| `/island rename [이름]`     | 섬 홈 이름을 변경합니다.      |

#### 🛠️ 섬 관리 기능 이용하기

섬과 관련된 여러 기능은 관리메뉴를 통해 더 편하게 이용할 수 있습니다.

관리메뉴에서는 섬 운영에 필요한 여러 기능을 한눈에 확인하고 빠르게 접근할 수 있습니다.

> 판테리움에서는 일부 기능을 메뉴 형태로 더 쉽게 이용할 수 있습니다.

#### 🔄 섬 초기화

섬을 새로 시작하고 싶을 때는 섬 초기화 기능을 사용할 수 있습니다.

**초기화 관련 주요 명령어**

| 명령어         | 설명             |
| ----------- | -------------- |
| `/is reset` | 자신의 섬을 초기화합니다. |

> ⚠️ 섬 초기화는 되돌리기 어려울 수 있으니 신중하게 진행해주세요.

### ⭐ 섬이 중요한 이유

판테리움에서는 여러 콘텐츠가 결국 섬과 연결됩니다.

* 자원을 모으면 섬을 더 효율적으로 운영할 수 있고
* 섬을 정리하면 생활 콘텐츠가 더 편해지고
* 꾸미기를 통해 나만의 개성을 표현할 수 있고
* 팀원과 함께하면 더 빠르고 효율적으로 성장할 수 있고
* 방문과 좋아요를 통해 다른 유저와의 교류도 가능해집니다

그래서 섬은 단순한 시작 지점이 아니라,\
**생활 / 성장 / 꾸미기 / 교류가 모두 이어지는 핵심 공간**입니다.
